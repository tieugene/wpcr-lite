WP Custom Repo Lite enables to use customer repository to update plugins and/or themes.

== Description ==

Simples way to make your plugins/themes updatable from your own repo:

1. Get web-site (it may be static)
2. Upload 2 files there
3. Add one command into plugin/theme code
4. Add one constant into `wp-config.php`
5. Enjoy

Use 'README' link below.
